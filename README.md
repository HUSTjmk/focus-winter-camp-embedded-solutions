# 机电/嵌入式赛道的参考解决方案--这个是给xdx看的，examples,datasheets放历程和资料

## 简介
- 你需要与其余两人合作，在一周的时间内，用我们提供的板子和模块，参考我们提供的资料，制作一个完整的桌宠。你需要独立完成3D外壳的设计和打印、板子的焊接、以及代码的编写与调试
